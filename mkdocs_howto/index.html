    darkmodeの切り替え#

    下の設定でできる

    theme:
palette:
    - scheme: default
      toggle:
          icon: material/toggle-switch-off-outline
          name: ダークモードに切り替えます。
    - scheme: slate
      primary: red
      accent: red
      toggle:
          icon: material/toggle-switch
          name: ライトモードに切り替えます
